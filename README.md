# Control for usb missile launcher for linux
![Missile](https://github.com/tsydd/missile/blob/master/img/missile.jpg)

### Requirements
- libusb-dev
- libgtkmm-3.0-dev

### Build
```bash
cmake --build .
```

### Run
```bash
sudo ./missile
```
    
### Controls
w | move up
s | move down
a | move left
d | move right
space | fire
